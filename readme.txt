how to use:

	how to setup:
	  1. activate the virtualenv 
	  2. install the required libs. this can be found at requirements.txt
	  3. got to task app and run runserver command


	how to setup the project in windows.
	   1, extract project folder somewhere in your computer
	   2, via cmd, cd to "..\project\env\scripts"
	   3, activate the virtualenv by running the command activate
	   4, abck to the project root dir and cd to task
	   5, install required lib and from requirements.txt file
	   6, run command "python manage.py runserver" to start the app
	   7, open browser and togo the localhost url "127.0.0.1:8000"
